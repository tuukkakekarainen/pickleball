Tools used and prompts: Antigravity / Sonnet 4.6
Prompt 
Create a simple Pickleball game for one or two players with HTML5 and JavaScript embedded inside it.
There should be three difficulty levels: Kids, Senior Citizens, and Masters.


There should be also "Rules" section that shows up the real Pickleball rules and court layout.

When Kids difficulty level selected the ball is easy to reach but you need to run a lot.
Senior Citizens is super easy and there is not much ball travel.
Masters is the ultimate game mode only for pros.

Use different accuracy for each levels for backhand and forehand shots and serves.
Use fun but realistic sound effects for serving and hitting the ball. Also if ball is missed or out of court there should be sound effect.
Play effect when losing or winning.


Use vivid typography and colour palette for Kids, antique one for Senior Citizens and professional one for Masters.

Keep everything in one HTML5 file (CSS can be embedded in <style>)
Generate the complete HTML file including:

<!DOCTYPE html>
<html>, <head>, and <body> sections
All required Bootstrap CDN links
Embedded CSS and JavaScript if needed
